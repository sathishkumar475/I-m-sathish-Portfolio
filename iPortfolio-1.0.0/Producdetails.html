ðŸ’¡ Project Overview

The Employee Productivity Tracker is a comprehensive React web application designed to track daily employee activities, team productivity, and performance metrics inside an organization. It supports role-based access so that Admins, Managers, Team Leaders, and Users each see dashboards and features relevant to their responsibilities.

Employees log their daily work in the form of activities with predefined point values. Team Leaders verify these activities, and the system generates daily and monthly reports, rankings, and statistics for both employees and teams.

ðŸ§± Project Structure & Tech Stack

The application is structured as a typical React SPA with clean separation of components, pages, context, services, and styles.

  • Frontend: React 18, functional components
  • Routing: React Router v6 for multi-role dashboards
  • State Management: React Context API for authentication and role-based access
  • HTTP Client: Axios with a central api.js service layer
  • Date Handling: date-fns for formatting and monthly navigation
  • Styling: Custom responsive CSS modules for dashboards and reports
