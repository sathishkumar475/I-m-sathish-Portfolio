SpiceGarden is a lightweight, browser-based billing system built for speed and simplicity. It allows staff to add menu items to a cart with a single click, manage menu items, accept payments via QR confirmation, print bills, and view monthly sales reports ‚Äî everything stored locally.

  1. Open index.html in a modern browser (supports ES6 modules).
